# 哈莉·奎茵 第一季

DC 原创动画剧集，该剧主创为《无能为力》班底 Justin Halpern & Patrick Schumacker 组合和 Dean Lorey。这是这个 DC 新流媒体平台开出绿灯的第三部剧集，前两部是真人版《泰坦》和《少年正义联盟》第 3 季。

IMDB 8.4 分，Quinn 的配音为生活大爆炸 Penny 的饰演者 Kaley Cuoco。

个人感觉比真人版电影《自杀小队》要精彩很多。
