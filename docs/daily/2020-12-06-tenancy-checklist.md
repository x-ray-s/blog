# {{$page.title}}

### 💰 价格

- 月租
- 支付方式 (押一付三/押一付一等)
- 中介费 / 服务费
- 水 电 燃气 供暖 网费

以最常见的押一付三，中介费为一个月方式来计算，那么你第一次租房需要支付 5 个月房租。

比较几个房租价格的时候要考虑的是 **中介费 + 房租 + 可选费用（网费 供暖）**，也可以除以 12，按照月租金来比较。

### 📍 位置

- 通勤距离
- 通勤费用
- 通勤时间成本

也可以把通勤的费用，作为比较租房成本的一个因素。

### 居住舒适度

- 楼层、是否有电梯、电梯与住户比
- 装修程度
- 卧室大小 朝向
- 公共区域
- 室友/邻居职业 作息
- 噪音情况 （临街、地铁、机场、市场、装修）

通过整体舒适度来比较房屋。

### 周边设施

- 小区停车费用
- 小区环境
- 周围配套设施 （超市、医院、商场）
- 物业服务
