# This

### 问题： 请解释 JavaScript 中 this 是如何工作的。

1. 由 `new` 调用？绑定到新创建的对象
2. 有 `call` 或者 `apply` （或者 `bind`）调用？绑定到指定的对象
3. 由上下文对象调用？绑定到哪个上下文对象
4. 默认：在严格模式下绑定到 `undefined`，否则绑定到全局对象
