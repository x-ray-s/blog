### 如何修改 person 对象

```js
const person = { name: "Lydia Hallie" };

Object.seal(person);
```

- A: person.name = "Evan Bacon"
- B: person.age = 21
- C: delete person.name
- D: Object.assign(person, { age: 21 })

> `Object.seal()` 方法封闭一个对象，阻止添加新属性并将所有现有属性标记为不可配置。当前属性的值只要原来是可写的就可以改变。
